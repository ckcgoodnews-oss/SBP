# SBP Enterprise 1.0.0 Administration Release Notes

## Goal
Complete Administration before CRM expansion.

## Included
- User management API/UI foundation
- Roles and permissions API/UI foundation
- Departments and locations API/UI foundation
- Sessions, API keys, service accounts
- Security dashboard metrics
- Audit timeline
- Impersonation and MFA administration pages
- Standardized API responses and errors
- Shared admin table/card/button components

## Acceptance Criteria
- `npm run typecheck` passes
- `npm run build` passes
- All `/admin/...` pages render
- All `/api/admin/...` routes return `{ ok: true, data: ... }` or actionable error JSON
