# Upgrade Guide — SBP Enterprise 1.0.0 Administration

1. Back up project.
2. Unzip package into project root.
3. Confirm `.env.local` has Supabase service role key.
4. Run `npm run typecheck`.
5. Run `npm run build`.
6. Run `npm run dev`.
7. Test admin pages.

No SQL migrations are included in this release.
